# Bash Scripting Labs

This repository contains a collection of Bash scripts for hands-on lab exercises and challenge tasks.

## Project Contents

- `Challenge1_menu_tool.sh` - An interactive menu-driven script to demonstrate Bash menu options and user interaction.
- `Challenge3_infrastructure_setup.sh` - A script for infrastructure setup tasks, likely provisioning or configuration automation.
- `Lab1_nginx_recovery.sh` - A recovery script for restoring or restarting an Nginx service.
- `Lab3_ec2_monitoring.sh` - A monitoring script intended for AWS EC2 instances.
